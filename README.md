

#Proje Açıklaması

Bu proje, kullanıcıların görevlerini kolayca yönetmesini sağlayan bir web uygulamasıdır.
Kullanıcılar sisteme kayıt olabilir, giriş yapabilir ve kendi görevlerini ekleyip düzenleyebilir veya silebilir.
Ahi Evran Tanıtım Web Sitesi
Bu proje, Ahilik teşkilatının kurucusu olan Ahi Evran'ın hayatını, eserlerini ve Ahilik kültürünü tanıtmak amacıyla hazırlanmış bir web sitesidir.

##Proje Amacı

Bu web sitesi ile kullanıcıların:
Ahi Evran'ın hayatını öğrenmesi
Ahilik teşkilatı hakkında bilgi edinmesi
Ahiliğin önemi ve özelliklerini anlaması
amaçlanmaktadır.

##Hedef Kullanıcılar

Öğrenciler
Günlük görevlerini takip etmek isteyen kullanıcılar
Görev yönetimi yapmak isteyen herkes
